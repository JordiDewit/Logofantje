## Logofantje

# Wat is Logofantje

Logofantje is een webapplicatie waar Yna Bauwens als studente Logopedie aan de Arteveldehogeschool, zelfgemaakt therapiemateriaal kan op plaatsen. Zij kan therapiemateriaal toevoegen, wijzigen en deleten. De bedoeling is dat gebruikers die ingelogd zijn via hun Facebook account, de pdf van het materiaal kunnen downloaden om dit materiaal ook na te maken. Het gaat trouwens over educatief materiaal.
